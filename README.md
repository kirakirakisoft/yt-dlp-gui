# yt-dlp GUI v3.0

**キラキラキソフト / KiraKiraKi Soft**

コマンド入力なしで [yt-dlp](https://github.com/yt-dlp/yt-dlp) を使うための、シンプルなGUIフロントエンドです。
「yt-dlpを使いたいけど、毎回コマンドを打つのは面倒・難しい」という方向けに作りました。

[English README is here → README_en.md](README_en.md)

---

## できること

- URLを貼ってボタンを押すだけでダウンロード
- **動画+音声(mp4)** / **音声のみ(mp3)** の切り替え
- **時間指定ダウンロード**(必要な範囲だけ保存)
- **保存先フォルダの選択**(前回の設定を記憶します)
- 動画は **H.264(avc1)を最優先** で取得 — DaVinci Resolve などの編集ソフトでそのまま読み込める形式を狙います
- メンバー限定・年齢制限動画のための **ブラウザCookie連携**(任意・初期設定はOFF)
- リアルタイムのログ表示、ダウンロードの中止ボタン
- **アイコン画像のカスタマイズ** — `icon.png` を同じフォルダに置くと、ヘッダーとタスクバーに表示されます(**80x80以下の正方形PNG推奨** — そのままのサイズで表示されるため最も綺麗です。大きい画像は自動縮小されますが画質が粗くなる場合があります。なくても動きます)

## 必要なもの

| ソフト | 必須? | 用途 |
|---|---|---|
| Python 3.9以降 | 必須 | このアプリの実行 |
| yt-dlp | 必須 | ダウンロード本体 |
| ffmpeg | 必須 | 映像と音声の結合、mp3変換 |
| Node.js | 任意 | 一部の動画の解析(なくても多くの動画は落とせます) |

### インストール手順(Windows)

1. **Python** — [python.org](https://www.python.org/downloads/) からインストール。
   インストーラー最初の画面で **「Add python.exe to PATH」に必ずチェック** を入れてください。
2. **yt-dlp** — コマンドプロンプトを開いて:
   ```
   pip install -U yt-dlp
   ```
3. **ffmpeg** — コマンドプロンプトで:
   ```
   winget install Gyan.FFmpeg
   ```
   (実行後、PCを再起動するか新しいコマンドプロンプトを開き直してください)
4. **Node.js(任意)** — 一部の動画でエラーが出る場合のみ:
   ```
   winget install OpenJS.NodeJS.LTS
   ```

### 起動方法

`yt_dlp_gui.py` をダブルクリック、またはコマンドプロンプトで:

```
python yt_dlp_gui.py
```

※ ダブルクリック時に黒いウィンドウ(コンソール)を出したくない場合は、
ファイル名を `yt_dlp_gui.pyw` に変更してください。

起動時に yt-dlp / ffmpeg / Node.js を自動チェックし、足りないものはログ欄でお知らせします。

## 使い方

1. 動画のURLを貼り付ける
2. モード(動画 / 音声)を選ぶ
3. 必要なら時間指定をONにして範囲を入力(`HH:MM:SS` 形式)
4. 「ダウンロード開始」を押す

保存先は「変更...」ボタンでいつでも変えられます。設定は自動で記憶されます。

## ブラウザCookie機能について(重要)

メンバー限定動画や年齢制限付き動画など、**ログインが必要なコンテンツ**をダウンロードする場合のみ使う機能です。初期設定はOFFです。

- **Firefox推奨**:Chrome系ブラウザは近年Cookieの暗号化が強化されており、
  外部ツールからの読み取りに失敗しやすいためです。Firefoxで対象サイトに
  ログインした状態でONにしてください。
- **自己責任でお使いください**:ログイン状態でのダウンロードツール使用は、
  プラットフォーム側に自動化ツールの利用として検知され、**アカウントに
  制限がかかる可能性がゼロではありません**。心配な場合はメインアカウント
  ではなくサブアカウントの使用を検討してください。
- このアプリがCookieを外部に送信することはありません。Cookieは
  お使いのPC内で yt-dlp に渡されるだけです。

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 「yt-dlp が見つかりません」と出る | `pip install -U yt-dlp` を実行。それでも出る場合はPythonインストール時のPATH設定を確認 |
| 映像と音声がバラバラに保存される / mp3にならない | ffmpegが未導入です。上記手順でインストール |
| `ERROR: Sign in to confirm...` 系のエラー | Cookie機能をONにして再試行(Firefoxでログインしておくこと) |
| 一部の動画だけ `ERROR` になる | Node.jsを導入すると解決する場合があります。また `pip install -U yt-dlp` で最新版への更新も試してください |
| 文字化けしたファイル名になる | ご利用の環境のシステムロケールをUTF-8に設定すると改善する場合があります |
| 昨日まで動いていたのに急に落とせなくなった | プラットフォーム側の仕様変更の可能性が高いです。まず `pip install -U yt-dlp` で更新してください |
